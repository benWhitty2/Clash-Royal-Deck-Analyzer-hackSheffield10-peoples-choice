# hackSheffield10

This project was a collaboration between me and a team mate who worked through paired programming and agentic workflows to develop a classifier, training data and GUI that can be seen in this project. 

The aim of the project is to provide an intermediate clash royal player instant feed back on the viability and archetype of a deck that they have assembled. The project uses Pytorch for the classifier, featuring a sequential neural network ending in a softmax layer, and the GUI uses Tkinter to give an intuitive interface that can be used regardless of previous experience.
